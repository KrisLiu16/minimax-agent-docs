## 一般问题

### Agent 支持哪些语言？

目前支持中文和英文的对话。知识库文档可以是中文或英文，Agent 会根据用户的提问语言自动匹配。

### 知识库文档有大小限制吗？

单篇文档建议不超过 10000 字。过长的文档建议拆分为多个小文档，每篇聚焦一个主题，有助于提升检索准确率。

### 同步频率是多少？

自动同步每 30 分钟执行一次。也可以在 admin 后台手动触发即时同步。

## 知识库相关

### 如何让 Agent 更准确地回答？

1. **文档标题要精确** — 标题应清晰描述文档内容
2. **内容要结构化** — 使用标题、列表等 Markdown 格式
3. **一篇文档一个主题** — 避免一篇文档涵盖太多内容
4. **使用 AI 增强** — 在编辑文档时使用 AI 增强功能生成搜索元数据

### 文档更新后多久生效？

手动同步后立即生效。文档会同时写入数据库和 Elasticsearch，无需等待。

### 删除文档后知识库还能搜到吗？

不会。删除操作会同步清理数据库和 Elasticsearch 中的记录。

## 技术问题

### 支持哪些文件格式？

目前同步系统支持 `.mdx` 格式。`.md` 文件不会被自动同步，但可以在 admin 后台手动创建文档。

### 如何处理多语言文档？

建议按语言分目录管理：

```
docs/
├── zh/          ← 中文文档
│   ├── intro.mdx
│   └── guide.mdx
└── en/          ← 英文文档
    ├── intro.mdx
    └── guide.mdx
```

同步时分别指定子目录和分类名即可。
