## 功能概述

MiniMax Agent 支持实时语音通话，用户可以通过语音与 Agent 进行自然对话。

## 支持的场景

- **电话客服** — 用户拨打客服热线，由 Agent 接听并处理
- **语音助手** — 在 App 内集成语音交互功能
- **外呼回访** — Agent 主动拨打用户电话进行回访

## 语音参数配置

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 语音模型 | TTS 语音合成模型 | speech-02-hd |
| 语速 | 语音播放速度 | 1.0 |
| 音色 | 预设音色或自定义音色 | 默认女声 |
| VAD 灵敏度 | 语音活动检测灵敏度 | 中 |

## 最佳实践

1. **保持回复简洁** — 语音场景下长篇回复体验不佳，建议知识库文档的回答要点控制在 3 句以内
2. **避免复杂格式** — 表格、代码块等在语音场景下无法呈现，用口语化表述
3. **设置打断策略** — 允许用户随时打断 Agent 的回复，提升交互体验
