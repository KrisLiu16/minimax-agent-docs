## Supported Platforms

| Platform | Status | Notes |
|----------|--------|-------|
| Web (agent.minimaxi.com) | ✅ | Primary access point |
| Mobile App (MiniMax official app) | ✅ | Synced with web |
| Lark | ✅ | Group bot supported |
| WeCom (Enterprise WeChat) | ✅ | Group bot supported |
| Personal WeChat | ✅ | Fully rebuilt in 03.27 update |
| PC Desktop client | ❌ | MaxClaw not supported — use web |

## Lark / WeCom Setup (5 Steps)

1. Tell your lobster "I want to connect Lark" (or WeCom)
2. Click the link the lobster provides and create a bot
3. Send a message to the bot to get a pairing code
4. Give the pairing code to your lobster
5. Done!

See the MaxClaw User Guide for detailed step-by-step screenshots.

## WeChat Setup

- **New lobsters:** Type "connect WeChat" in the chat box → one-click setup
- **Existing lobsters:** Automatic sandbox upgrade in progress — no rebuild needed

## Adding a Bot to a Group Chat

Group chat → Group Settings → Group Bots → Add → Find MaxClaw → Confirm

## Multi-Lobster + IM Combination

Deploy up to 3 lobsters, connect each to a different IM bot, add them all to the same group. Mention everyone to get all lobsters working in parallel.

## Notes

- Don't manually update MaxClaw — it can corrupt the container
- IM connections may drop after a version update and need to be reconfigured
- On connection failure, try Auto Repair first before deleting and rebuilding
