## About MiniMax Agent

MiniMax Agent is an AI assistant platform by MiniMax, accessible at [agent.minimaxi.com](https://agent.minimaxi.com). It includes four core products:

| Product | Role | Description |
|---------|------|-------------|
| **Agent Chat** | AI conversation | Multi-turn dialogue with knowledge base retrieval |
| **MaxClaw** | Cloud AI assistant | Cloud-hosted OpenClaw instance, 7×24 uptime, no setup needed |
| **SkillHub** | Skill community | Hundreds of ready-made skills, install into MaxClaw with one click |
| **Desktop** | Desktop app | Deploy OpenClaw locally on your own machine |

## Quick Start

### Using MaxClaw (Recommended for new users)

1. Go to [agent.minimaxi.com](https://agent.minimaxi.com) and log in
2. Open the MaxClaw page, click "Create" to deploy your personal instance
3. Start chatting immediately — no API Key or server required

### Expanding with SkillHub

1. Browse the SkillHub community for skills
2. Click "Use Now" on any skill you want
3. The skill is instantly available in your MaxClaw conversation
