## What is SkillHub?

SkillHub is the **skill marketplace** for MiniMax Agent, offering hundreds of ready-made skills you can install into MaxClaw (your lobster) with one click.

Install a skill → your lobster gains that capability. **No coding required.**

## How to Use

1. Open the SkillHub community and browse the skill catalog
2. Click "Use Now" on any skill you like
3. The skill is immediately available in your MaxClaw or MiniMax Agent conversation — no configuration needed

## Office Skills (Recommended)

MiniMax Office Skills is fully open-source (MIT license, free to use and modify), covering four key scenarios:

- **Word** — Lossless editing, zero format loss
- **Excel** — Native formula preservation, pivot table support
- **PPT** — Professional-grade layouts
- **PDF** — End-to-end delivery in a single sentence

## Create Skills and Earn Credits

You can build your own Skill and publish it to SkillHub:

- **Earn 100 credits every time someone uses your skill**
- High-quality skills can generate ongoing passive credit income

## Notes

Installing skills consumes credits (the model needs to understand, configure, and execute each skill). If credits run out, skills will stop working until you recharge.
