## Self-Check Order

When something goes wrong, follow these steps in order:

1. **Check credit balance** — depleted credits cause most errors
2. **Click Restart** — top-right restart button
3. **Send /new** — clear chat history and start a new conversation
4. **Auto Repair** — top-right gear → Auto Repair
5. **Delete and recreate** — if all else fails, delete MaxClaw and create a new one
6. **Contact support** — provide chat_id and email MiniMaxAgent@minimax.io

> chat_id: Copy the number at the end of the URL, e.g. `https://agent.minimaxi.com/chat?id=370943808623550` → `370943808623550`

---

## Error Reference

### 400 / HTTP 500 / Internal Server Error

**Cause:** Credits exhausted (most common), or service error.

**Fix:**
1. Check credit balance
2. Credits sufficient → click Restart
3. Still failing → provide chat_id

---

### 413 / Maximum Request Body Size Exceeded

**Cause:** Conversation history too long (approx. >1MB). Common after image recognition.

**Fix:** Send `/new` to clear history, then retry.

---

### Moderation Blocked / Input Box Disabled

**Cause:** Content triggered the platform's content safety review.

**Fix:** Delete MaxClaw and redeploy. If you believe it's a false positive, email MiniMaxAgent@minimax.io with your chat_id — credits will be refunded for confirmed false positives.

---

### Task Stopped / "Try a New Task"

**Cause:** Safety review triggered.

**Fix:** Delete and recreate MaxClaw.

---

### Stuck on Loading / No Response

**Fix:**
1. Click Restart (top-right)
2. Click Auto Repair
3. Still failing → provide chat_id

---

### Response Timeout

**Fix:** Send `/new` to start a new conversation — usually recovers immediately.
