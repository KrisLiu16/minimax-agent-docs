## Lark

### Setup Flow (5 Steps)

1. Tell your lobster "I want to connect Lark"
2. Click the link the lobster provides and create a Lark bot
3. Send a message to the bot to get a pairing code
4. Give the pairing code to your lobster
5. Setup complete

### Pairing Keeps Failing / Stuck on Restarting / Service Busy

**Cause:** Unclear prompts caused MaxClaw to corrupt its own configuration file.

**Fix:**
1. Top-right gear → Auto Repair
2. If that fails → delete and redeploy (avoid making frequent configuration changes)

### Thread/Topic Replies Going to Main Group Chat

**Cause:** Lark thread routing parameters not configured correctly. Contact support with your chat_id.

### Adding the Bot to a Group

1. Open the target group chat
2. Group Settings → Group Bots → Add → Find MaxClaw → Confirm

## IM Connection Lost After a Version Update

If your IM connection drops after an update, reconnect manually:
1. Open the MaxClaw web conversation to check memory and IM connection status
2. Follow the setup flow again for the relevant platform

Affected users will receive credit compensation. If you haven't received it, provide your chat_id and contact MiniMaxAgent@minimax.io.
