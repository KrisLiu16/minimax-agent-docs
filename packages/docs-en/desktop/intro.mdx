## What is the Desktop App?

MiniMax Agent Desktop is a **desktop client application** that lets you deploy OpenClaw locally on your own machine, keeping full control of your data.

> Note: Desktop is for local OpenClaw deployment — different from MaxClaw (cloud-hosted).

## Key Features

- **Local OpenClaw deployment** — Run your AI assistant on your own computer
- **Full data ownership** — All conversations and files stay on your machine
- **Built-in "Lobster Doctor"** — When deploying via Desktop, safety diagnosis skills are pre-installed:
  - Automatically blocks high-risk operations (e.g., accidental deletion of important files)
  - Auto-diagnoses and attempts to repair errors when they occur
  - If it can't fix the issue, it explains the cause and suggests next steps

## Desktop vs MaxClaw

| | Desktop (Local OpenClaw) | MaxClaw (Cloud) |
|--|--|--|
| Where it runs | Your local machine | MiniMax cloud servers |
| Data storage | Local | Cloud |
| Requires API Key | ✅ Yes (MiniMax Open Platform) | ❌ No — works out of the box |
| 7×24 background operation | Requires PC to be on | ✅ Always on |
| Billing | Credits (same as Agent Chat) | Credits |

> **Note:** Desktop and Agent Chat share the same billing method — both consume Credits. The difference is functionality and where they run, not how they bill.

## About Billing

Desktop uses an API Key (from platform.minimaxi.com) for authentication, and billing is in **Credits** — the same as Agent Chat Mode. Manage your API Key on the MiniMax Open Platform.
