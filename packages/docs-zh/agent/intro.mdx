## 什么是 Agent 对话模式？

Agent 对话模式是 MiniMax Agent 平台的核心 AI 对话功能，支持多轮上下文对话，并可结合知识库实现精准问答。

入口：[agent.minimaxi.com](https://agent.minimaxi.com)

## 核心能力

- **多轮上下文对话** — AI 理解对话历史，连续追问、深入讨论
- **知识库检索增强（RAG）** — 基于结构化知识库文档回答，结果更准确可靠
- **工具调用** — 支持联网搜索、图片生成、文件读写、代码执行等
- **多模态支持** — 支持文本、图片输入

## 与 MaxClaw 的区别

| | Agent 对话模式 | MaxClaw |
|--|--|--|
| 定位 | 即时 AI 对话 | 云端长期运行的个人 AI 助手 |
| 长期记忆 | 无（会话结束即重置） | ✅ 有（MEMORY.md 持久化） |
| 定时任务 | 无 | ✅ 支持 Cron |
| IM 接入 | 无 | ✅ 飞书/企微/微信等 |
| 计费 | 按 token 消耗积分 | 按 token 消耗积分 |

## 使用建议

- 快速问答、单次任务 → 使用 Agent 对话模式
- 需要长期记忆、自动化、IM 接入 → 使用 MaxClaw

## 对话上下文管理

- 每轮对话携带历史上下文，上下文越长消耗积分越多
- 开启新对话（/new）可清除当前会话历史，降低 token 消耗
