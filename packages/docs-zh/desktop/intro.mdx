## 什么是 Desktop？

MiniMax Agent Desktop 是**桌面端客户端应用**，支持在本地电脑部署 OpenClaw，数据完全由自己掌控。

> 注意：Desktop 用于本地部署 OpenClaw，与 MaxClaw（云端版）是不同的使用方式。

## 主要功能

- **本地部署 OpenClaw** — 在自己的电脑上运行 AI 助手，无需依赖云端
- **自主掌控数据** — 所有对话、文件存储在本地
- **自带"龙虾医生"** — 通过 Desktop 部署本地 OpenClaw 时，会预装安全诊断技能：
  - 自动拦截高危操作（如误删重要文件）
  - 遇到报错时自动诊断 + 尝试修复
  - 修不好的问题会说明原因和建议方案

## Desktop 与 MaxClaw 的区别

| | Desktop（本地 OpenClaw） | MaxClaw（云端） |
|--|--|--|
| 运行位置 | 本地电脑 | MiniMax 云端服务器 |
| 数据存储 | 本地 | 云端 |
| 需要自备 API Key | ✅ 是（接入 MiniMax 开放平台） | ❌ 否，开箱即用 |
| 7×24 后台运行 | 依赖电脑开机 | ✅ 是 |
| 计费方式 | 积分（Credits），与 Agent 对话模式相同 | 积分（Credits） |

> **注意：** Desktop 与 Agent 对话模式计费方式相同，均消耗积分（Credits）。区别仅在于功能和运行位置，不是计费方式。

## 关于积分

Desktop 本地部署 OpenClaw 接入 MiniMax 模型，同样消耗**积分（Credits）**，与 Agent 对话模式一致，使用 API Key 鉴权。请在 MiniMax 开放平台（platform.minimaxi.com）管理 API Key。
