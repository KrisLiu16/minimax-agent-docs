## 产品定位

### 客服机器人能回答什么问题？

主要解答产品使用、功能咨询、积分计费、技术报错等问题。**不提供**：图片/视频生成、闲聊、与产品无关的内容。

### MaxClaw 和 OpenClaw 是什么关系？

MaxClaw 是云端托管版的 OpenClaw，底层由 OpenClaw 驱动，接入 MiniMax 自有模型。如需本地部署，可自行安装 OpenClaw（参考 openclaw.ai）。

### MiniMax Agent 积分和 MiniMax 开放平台余额通用吗？

**不通用。** 两者是完全独立的账号体系：

| | MiniMax Agent（MaxClaw） | MiniMax 开放平台 |
|---|---|---|
| 入口 | agent.minimaxi.com | platform.minimaxi.com |
| 计费单位 | 积分（Credits） | 人民币余额 |
| 用途 | 使用 MaxClaw | 调用 API / 本地部署 |

## 账号与登录

### 手机 APP 和网页端数据互通吗？

互通。MaxClaw 网页端和 MiniMax 官方手机 APP 使用同一账号，数据、积分、对话记录均共享。

### 如何提供 chat_id 供排查？

打开对话页面，复制链接末尾数字即为 chat_id。
示例：`https://agent.minimaxi.com/chat?id=370943808623550` → chat_id 为 `370943808623550`

⚠️ 遇到技术问题，客服需要 chat_id 才能进一步排查，请提前准备。
