## 同步失败

### git pull 失败

**现象**: 同步日志显示 `git pull 失败，中止同步`

**原因与解决**:
- **认证失败** — 检查 `GITHUB_TOKEN` 环境变量是否正确设置
- **网络问题** — 确认服务器能访问 github.com
- **合并冲突** — 不要直接修改服务器上的本地仓库，所有修改通过 git push 提交

### RPC 调用失败

**现象**: 同步日志显示 `新增失败` 或 `更新失败`，错误信息包含 RPC

**原因与解决**:
- **chat-server 不可用** — 检查 chat-server 服务是否正常运行
- **参数错误** — 检查 config_id 是否有效
- **文档内容过大** — 单篇文档内容不宜超过 50KB

## 知识库搜索不到文档

### 文档已同步但搜不到

**可能原因**:
1. ES 索引延迟 — 等待几秒后重试
2. 搜索关键词与文档内容差异太大 — 使用 AI 增强功能优化搜索元数据
3. 文档分类不匹配 — 确认搜索时使用的 config_id 与同步时一致

### 文档内容为空

**原因**: MDX 文件的 frontmatter 格式不正确，导致解析失败

**检查**:
- frontmatter 必须以 `---` 开头和结尾
- `title` 字段必须存在且非空
- 确认文件编码为 UTF-8
