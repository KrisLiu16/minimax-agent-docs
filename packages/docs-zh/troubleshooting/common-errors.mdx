## 快速自查步骤

遇到任何问题，请先按以下顺序排查：

1. **检查积分余额** — 右上角积分耗尽会导致大多数错误
2. **点击重启** — 右上角重启按钮
3. **发 /new** — 清除对话历史，开启新对话
4. **自动修复** — 右上角齿轮 → 自动修复
5. **删除重建** — 以上均无效时，删除 MaxClaw 后重新创建
6. **联系支持** — 提供 chat_id 发邮件至 MiniMaxAgent@minimax.io

> chat_id 获取：`https://agent.minimaxi.com/chat?id=370943808623550` 中末尾数字即为 chat_id

## 错误码速查

### 400 / HTTP 500 / Internal server error

**原因：** 积分耗尽（最常见），或服务异常。

**处理：**
1. 检查积分余额
2. 积分充足 → 点击重启
3. 仍无效 → 提供 chat_id

---

### 413 / Maximum request body size exceeded / 请求体过大

**原因：** 对话历史过长（约超 1MB），图片识别后尤其容易触发。

**处理：** 发送 `/new` 清除历史，然后重试。

---

### Moderation blocked / 内容被拦截

**原因：** 内容触发平台内容审核。

**处理：** 删除当前 MaxClaw，重新部署后重试。如认为误伤，发邮件至 MiniMaxAgent@minimax.io 附 chat_id 申请审核，误伤退还积分。

---

### Task stopped, try a new task / 任务停止无法输入

**原因：** 触发安全审核。

**处理：** 删除当前 MaxClaw 重新创建。

---

### loading 无响应 / 长时间不回消息

**处理：**
1. 右上角重启
2. 点击自动修复
3. 仍无效 → 提供 chat_id

---

### 发消息后超时

**处理：** 发送 `/new` 开启新对话，通常立即恢复。
