## /new 命令

`/new` 开启新对话，清除**短期记忆**（当前对话历史），不影响长期记忆（MEMORY.md、技能、配置等持久化内容）。

- 清除：当前对话轮次的聊天记录（不再随请求发送给模型）
- 保留：MaxClaw 记住的重要信息、设置、已安装技能

**推荐用法：** 每完成一件事就发 `/new`，再开始下一件事，可大幅节省积分消耗。

## 定时任务（Cron）

MaxClaw 支持定时自动执行任务，如每日新闻推送、定时提醒等。

- 积分耗尽时定时任务会停止，充值后自动恢复
- 如定时任务消失，可在网页端对话中重新配置

## IM 平台连接

### 支持平台
飞书 / 企业微信 / 微信（个人）/ 钉钉

### 飞书 / 企微接入流程（已简化至 5 步）
1. 告诉龙虾「我要连飞书」（或企微）
2. 点击龙虾提供的链接，创建机器人
3. 向机器人发消息获取配对码
4. 把配对码告诉龙虾
5. 完成！

### 添加机器人到群聊
1. 打开目标群聊 → 群设置 → 群机器人 → 添加 → 找到 MaxClaw → 确认

### 注意事项
- 飞书子话题/线程回复：如出现回复到主群聊而非子话题，属于路由问题，可联系支持处理
- 连接失败可先尝试自动修复（右上角齿轮）；若仍失败，删除后重新部署，避免频繁修改配置
