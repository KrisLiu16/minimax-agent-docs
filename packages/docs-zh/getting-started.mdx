## 欢迎使用 MiniMax Agent

MiniMax Agent 是一款智能客服助手，支持文本对话和语音通话，帮助企业高效处理客户咨询。

## 接入流程

1. **创建客服配置** — 在 admin 后台「配置管理」tab 中创建一个客服配置
2. **配置知识库** — 添加业务相关的知识文档，Agent 会基于这些文档回答问题
3. **测试对话** — 在「会话管理」tab 中查看测试对话效果
4. **上线部署** — 确认效果后接入线上环境

## 核心概念

### 客服配置 (Config)

每个客服配置包含：
- 基本信息（名称、描述）
- 关联的知识库
- 对话参数（温度、最大轮次等）

### 知识库文档

知识库是 Agent 回答问题的依据。文档会被索引到 Elasticsearch 中，支持语义检索。

- 支持手动创建和编辑
- 支持从 GitHub 仓库批量同步
- 支持 AI 增强搜索元数据

### 会话 (Session)

每次用户与 Agent 的对话构成一个会话，包含完整的消息历史和引用的知识库文档。
