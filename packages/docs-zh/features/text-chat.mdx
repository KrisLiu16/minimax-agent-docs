## MaxClaw 对话核心机制

MaxClaw 的每次对话背后都有大模型参与，携带完整上下文（历史对话 + 工具结果）生成回复。

## /new 命令

`/new` 清除**当前对话的短期记忆**（聊天历史），下一个任务从空白上下文开始。

- **清除：** 当前对话轮次的消息记录
- **保留：** MEMORY.md、已安装技能、配置等长期持久化内容

**推荐用法：** 每完成一件事就发 `/new`，可大幅减少 token 消耗。

## 上下文越长，积分消耗越多

- 历史消息越多，每次请求携带的 token 越多
- 图片识别、工具调用等会带入大量上下文
- 建议及时 `/new` 切换任务

## 支持的输入类型

- 纯文本消息
- 图片（图片内容会被带入上下文，注意 token 消耗）
- 文件（上传后 Agent 可读取处理）

## 工具能力

MaxClaw 内置以下工具能力：

| 工具 | 说明 |
|------|------|
| 联网搜索 | 获取实时信息 |
| 图片生成 | 文字描述生成图片 |
| 视频生成 | 文字描述生成视频 |
| 文件读写 | 读取/创建/修改文件 |
| 代码执行 | 运行 Python 等代码 |
| 定时任务 | Cron 自动化任务 |

## 文件管理

- 生成的文件默认在 `memory/` 目录（内部工作区）
- 告诉 Agent「把文件移到 WorkSpace 目录下」可在文件列表中找到
