## 支持的接入方式

| 终端/平台 | 状态 | 说明 |
|-----------|------|------|
| 网页端（agent.minimaxi.com） | ✅ | 主要使用入口 |
| 手机 APP（MiniMax 官方 App） | ✅ | 与网页端数据互通 |
| 飞书 | ✅ | 支持群聊机器人 |
| 企业微信 | ✅ | 支持群聊机器人 |
| 个人微信 | ✅ | 03.27 全面修复上线 |
| PC 桌面客户端 | ❌ | 暂不支持 MaxClaw |

## 飞书 / 企微接入（5 步完成）

1. 告诉龙虾「我要连飞书」或「我要连企业微信」
2. 点击龙虾给的链接，创建机器人
3. 向机器人发消息获取配对码
4. 把配对码告诉龙虾
5. 完成！

详细图文步骤见 MaxClaw 使用指南。

## 微信接入

- **新龙虾：** 对话框输入「连接微信」→ 一键完成
- **老龙虾：** 等待系统自动升级，无需手动重建

## 群聊中添加机器人

1. 打开群聊 → 群设置 → 群机器人 → 添加 → 找到 MaxClaw → 确认

## 多龙虾 + IM 组合玩法

一个账号最多部署 3 只龙虾，每只连接不同的 IM 机器人，加入同一群聊，@ 所有人可同时调动所有龙虾协作。

## 注意事项

- 手动更新版本可能导致容器损坏，建议等待官方推送
- 版本更新后 IM 连接可能断开，需重新配置
- 连接失败时优先尝试「自动修复」，再考虑删除重建
